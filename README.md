# Калькулятор (Vanilla JS)

## Короткое описание

Простой веб-калькулятор с графическим интерфейсом, написанный на чистом JavaScript с использованием ООП. Все вычисления выполняются на клиентской стороне.

## Ключевые моменты

- **Язык:** Vanilla JavaScript (ES6+)
- **Сборка:** Parcel (dev-сервер и бандлинг)
- **Архитектура:** ООП (класс `App` и модуль утилит)
- **Интерфейс:** HTML + CSS

## Функционал

- Базовые арифметические операции: сложение, вычитание, умножение, деление
- Возведение в квадрат (x²)
- Удаление последнего символа (⮨)
- Очистка поля ввода (C)

## Структура проекта

```
calculatorVanilaJS/
├── index.html          # Основной HTML-файл
├── script.js           # Точка входа (импорт и инициализация App)
├── scripts/
│   ├── app.js          # Класс App (рендер интерфейса и обработчики)
│   └── utils.js        # Вспомогательные функции (back, sqrt, base, calcBase)
├── styles/
│   └── main.css        # Стили калькулятора
├── package.json        # Зависимости и скрипты (Parcel)
└── dist/               # Собранные файлы (генерируется Parcel)
```

## Зависимости

- **Parcel** (dev): для запуска локального dev-сервера с hot reload

Установка:

```bash
npm install
```

## Локальный запуск

Запустите dev-сервер с автоматическим открытием браузера:

```bash
npm run serve
```

Приложение откроется в браузере на `http://localhost:1234` (или другом порту, который Parcel укажет).

## Деплой

Проект автоматически деплоится на **GitHub Pages** при пуше в ветку `main` благодаря GitHub Actions.

**URL проекта:**  
[https://denisunderonov.github.io/calculatorVanilaJS/](https://denisunderonov.github.io/calculatorVanilaJS/)

### Настройка деплоя

1. В настройках репозитория (Settings → Pages) выберите источник: **GitHub Actions**
2. Workflow автоматически соберёт проект через Parcel и задеплоит папку `dist/` на GitHub Pages

## Технические детали

- **Модульная структура:** используются ES6-модули (`import/export`)
- **Рендеринг:** динамическая вставка HTML через `innerHTML`
- **Логика:** состояние калькулятора (первое число и операция) хранится в объекте `state`
- **Операции:** обрабатываются в `utils.js` (back, sqrt, base, calcBase)
